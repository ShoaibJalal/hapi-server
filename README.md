# hapi-server

Creating basic Node.js server with Hapi framework

This is an app where client can perform CRUD operations on backend.

## Routes exist on API...:

GET
http://localhost:8000/
--
GET
http://localhost:8000/tasks
--
GET specific task
http://localhost:8000/tasks/id id: should be an integer number.
--
POST
http://localhost:8000/tasks
--
PUT
http://localhost:8000/tasks/id id: should be an integer number.
--
PATCH
http://localhost:8000/tasks/id id: should be an integer number.
--
DELETE
http://localhost:8000/tasks/id id: should be an integer number.
